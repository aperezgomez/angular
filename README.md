## Installation

1. Clone the repository:
    ```bash
    git clone <YOUR_REPOSITORY_LINK>
    ```

2. Navigate to the desired project:
    ```bash
    cd <PROJECT_DIRECTORY>
    ```

3. Install the dependencies:
    ```bash
    npm install
    ```

4. Run the application:
    ```bash
    ng serve
    ```
